One click solution to provisioning, deployment and migration of your apps using Ansible and Django

One Love

Requirements and development setup

  • VirtualBox
    • All boxes are tested in VirtualBox virtualization platform
  • Vagrant
    • To run all VirtualBox instances effortlessly, Vagrant is used to automate VM creation/provisioning/destruction
  • Ansible
    • Server setup and configuration is done through Ansible for One Love, as for all other applications

First run

Clone this repo, cd into it, and run vagrant up. The process will take a while and should give you fully configured VirtualBox VM instance. Execute vagrant ssh onelove and you'll be inside VM, ready to develop. Run ~/bin/runserver.sh and point your browser to Vagrant VM. Login with admin@example.com/Sekrit
